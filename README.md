## 功能
 - 读取客户端日志文件中的 url
 - 将获取到的6个月历史记录与本地记录合并（1.3版本更新后api只能获取6个月历史记录）
 - 导出抽卡记录为 json 和 excel (已适配 UIGF 格式)
 - 展示抽卡报告 [抽卡报告预览](抽卡报告.md) ，鼠标悬停在数量上即可查看对应星级详情
 - 记录历史 url，如果可用无需重新抓包
 - 可选抓包本机 api 请求
 - 可选手动输入 url
